## Nonprofessional

在演奏音乐时，不管使用声学乐器还是电子乐器，都有可能达到损伤听力的声压级，尤其是在声学环境活跃的小房间里进行演奏时。

在某些情况下，听觉听到的声音是人们没有预料到的，这是听觉产生幻觉的结果，这种现象与听觉系统分析声音信号的方式有关。
